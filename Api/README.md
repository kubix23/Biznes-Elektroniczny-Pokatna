## Użyta technologia
Jest to kod wykonany w Pythonie pozwalający zaimportować przy pomocy webservice kategorie i produkty. Format danych przystosowany jest do danych z scrapera. Skrypt importuje następujące cechy: 
- Kategoria:
    - Obraz
    - Nazwa
    - Opis
- Produkty:
    - Nazwa
    - Opis
    - Waga
    - Cena
    - Obraz
    - Cechy
    - Producenta
    - Ilość
